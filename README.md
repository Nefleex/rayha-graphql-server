# Getting started

`git clone https://github.com/Nekuin/rayha-graphql-server.git`

`cd graphql-server`

`touch .env `

## .env file

After setting up your database and databse user, fill out the environment variables

```
PORT=3001
DATABASE=mydatabase
DATABASE_USER=myuser
DATABASE_PASSWORD=mypassword
```
